alias(other: magma.core.variables.abstract.AbstractProcessInterfaceObject)¶

Establishes an ‘alias’ relationship between ‘this’ and ‘other’ AbstractProcessInterface object (like Vars or Ports) across sub process boundaries when an ‘outer’ interface object is made an alias of an ‘inner’ interface object. In this case the ‘outer’ object points to the ‘inner’ object. As a first step self.alias(other) determines whether ‘self’ or ‘other’ is the ‘inner’ or ‘outer’ interface object and then estblishes the directional relationship from the ‘outer’ to the ‘inner’.

If one variable aliases another variable, the compiler will use the initial value of the ‘outer’ variable to initialize the inner variable. Any initial value of the ‘inner’ variable will be ignored.
