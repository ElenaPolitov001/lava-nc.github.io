probe(var_name: str, indices: Union[None, List, numpy.ndarray] = None, t_start: int = 1, t_end: int = 1000000000000, dt: int = 1, buffer_size: int = 100000, is_interactive: bool = False) → lava.core.generic.probing.Probe¶
Parameters

  • var_name – Name of state to be probed.

  • indices – The indices of the tensor elements in the state to be probed.

  • t_start (int) – Time step at which probing starts.

  • t_end (int) – Time step at which probing ends.

  • dt (int) – Interval at which <state> is probed.

  • buffer_size (int) – The bufferSize of the probe.

  • is_interactive (bool) – If True, times eries plot will be updated interactively while process runs.

This creates a new probe and returns it.
