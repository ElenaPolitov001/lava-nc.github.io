Compiler¶

class lava.core.generic.compiler.AbstractCompiler¶

Abstract base class for backend-specific compiler.

A compiler consumes a hierarchy of processes, compiles them and returns an executor with executables for each compiled process.

abstract compile(process: lava.core.generic.process.AbstractProcess) → lava.core.generic.executor.AbstractExecutor¶

Compiles process and its sub-processes and returns AbstractExecutor.
