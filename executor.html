Executor¶

class lava.core.generic.executor.AbstractExecutor(executables: List[lava.core.generic.executable.AbstractProcessExecutable], var_map: Dict[AbstractVar, ExecVar], time_series_map: Dict[AbstractVar, AbstractTimeSeries])¶

Abstract base class for any executor that executes an AbstractProcess for a specific backend.

An executor is created during compilation of one or multiple processes by a compiler.

property executable¶

Returns top-level process executable.
