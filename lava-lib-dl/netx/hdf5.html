HDF5

HDF5 network exchange module.

class lava.lib.dl.netx.hdf5.Network(*args, **kwargs)

Generates a Lava process for the network described in hdf5 config.

Parameters

  • net_config (str) – name of the hdf5 config filename.

  • num_layers (int, optional) – number of blocks to generate. An integer valuew will only generate the first num_layers blocks in the description. The actual number of generated layers may be less than num_layers. If it is None, all the layers are generated. Defaults to None.

  • has_graded_input (bool, optional) – flag indicating the input spike type of input layer. Defaults to False.

Number of layers in the network.
